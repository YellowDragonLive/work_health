# 久坐健康助手 (Health Assistant) - Windows 版

一款为 Windows 用户打造的极简、沉浸式久坐提醒工具，旨在通过 25/5 分钟的科学循环（番茄钟变体），保护您的视力与脊椎健康。

## 🌟 核心特性

- **强制窗口提醒**：时间结束后触发全屏（置顶且独占）提醒，确保您不会忽略休息指示。
- **全局媒体联动**：提醒触发时自动暂停浏览器视频（Youtube/Bilibili 等）及 PotPlayer 播放，休息结束自动恢复。
- **智能活动检测**：
  - **锁屏暂停**：系统锁屏时计时自动暂停。
  - **闲置暂停**：检测到用户长时间（默认 20 分钟）离开电脑时自动暂停，归座后自动继续。
- **高度可定制**：
  - 支持自定义提醒音乐（mp3/wav）。
  - 支持实时修改工作时长（1-120 分钟）。
- **极简系统驻留**：
  - 托盘图标显示实时倒计时与完成轮次。
  - 支持开机自启动，完全后台运行，无任何控制台黑窗口干扰。
- **紧急处理机制**：全屏提醒状态下支持 `Esc` 键暂时隐藏窗口 15 秒，方便处理紧急事务后再休息。

## 🛠️ 安装要求

- Windows 10/11
- Python 3.8+
- 依赖库：
  ```bash
  pip install pygame pystray Pillow olefile
  ```

## 🚀 快速开始

1. **克隆代码**：
   ```bash
   git clone https://github.com/YellowDragonLive/work_health.git
   cd work_health
   ```
2. **初始化素材**：
   运行生成默认图标和提示音：
   ```bash
   python generate_assets.py
   ```
3. **运行程序**：
   ```bash
   python src/main.py
   ```

## 📖 使用说明

- **托盘菜单**：右键点击右下角图标可进行时长设定、音乐选择、重置计时及启/停开机自启动。
- **实时预览**：鼠标悬停在托盘图标上即可查看当前倒计时。
- **日志记录**：程序运行状态及任何异常将记录在根目录下的 `app.log` 中。
